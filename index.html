¿Quienes somos?

B-Ready es un proyecto de 4to año diseñando
por Ilan Repiso, Marco Canton, Alex Carpel y Matias Blanco
con el objetivo de facilitar la organizacion
de las tareas de uno para
realizarlas de manera mas eficaz
y evitando estres innecesario


¿De que se trata el proyecto?

La idea es crear una pagina web con un
calendario personal que permite ingresar tareas
y las organiza dependiendo diversos factores
de una manera que cada uno pueda saber
cuanto tiempo hay que invertir y cuando para resolverlo
